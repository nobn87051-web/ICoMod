# ICo Mod 

A Fabric client mod for some Wynncraft QoL. 
Made by ahjd.

Built on Minecraft 1.21.11 with Fabric Loom and Kotlin.

* Is still in BETA, lots of incomplete features and upcoming ones

# Website - https://icomod.xyz/
- A WIP website for submitting GIFs to include in the mod catalogue
- Has a Ticket system for requesting changes/reporting issues
