# ICo Mod

A Fabric client mod for Wynncraft. Made for [ICo] — the Idiot Company.

Built on Minecraft 1.21.x with Fabric Loom and Kotlin.

## Build

```
./gradlew build
```

The jar lands in `build/libs/`.

## Run in dev

```
./gradlew runClient
```
